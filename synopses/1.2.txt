Internet и WWW
------------------------------------------------------------------------------

Internet - глобальная сеть передачи данных
  Протоколы образуют стек (когда одни протоколы работают поверх других)
  Стек протоколов Internet (избранные):
    Протокол IP
      позволяет передавать данные между двумя машинами сети
      вводит глобальную адресацию устройств внутри сети
      работает с datagram (кусочками данных)
      не гарантирует надежную доставку данных (UDP?)
    Протокол TCP (поверх IP)
      строго гарантированная доставка
      время доставки не гарантируется
      данные если протокол отвечает что данные доставлены, то это точно так
    Протоколы HTTP, FTP, P2P, SSH (поверх TCP)

WWWW
  1) WWW - множество гипертекстовых (взаимосвязанных) документов интернета (подмножество internet)
  2) WWW - множество протоколов передачи гипертекстовых документов
  Гиперссылка - способ указания связи двух документов
  
MIME - типы
------------------------------------------------------------------------------
  Возникли для передачи файлов в виде e-mail, где все представленно в виде теста.
  Именно MIME-тип определяет как и чем обрабатывать файл (не расширение файла, а именно MIME-тип).
  
  XML (text/xml) - это более строгий (в сравнении с HTML) формат описания документа
    Допускает разные форматы записи (поля объекта - вложенные теги объекта или поля объекта - атрибуты тега объекта)
  JSON (application/json) - совокупность массивов ([ "значение" ]) и словарей ({ "ключ" : "значение" }) на языке JS
    Позволяет легко представить дерево в виде тествого файла.
    
  Документы
    Статические - как правило, файл
    Динамические - генерируется на лету в application server по запросу пользователя
