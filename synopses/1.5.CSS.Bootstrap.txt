BOOTSTRAP
--------------------------------------------------------------------
  Bootstrap - готовая библиотека стилей (CSS-фраймворк) от компании Twitter.
  Позволяет разрабатывать приемлемый дизайн при базовых знаниях CSS.

  Что включает в себя Bootstrap?
    - Шаблон страниц
    - Сетка
    - Современные «стили по умолчанию»
    - Верстка: таблицы, формы, списки, кнопки, …
    - Компоненты: навигация, меню, пагинатор, …
    - JavaScript плагины

Сетка Bootstrap (самый большой профит от использования Bootstrap):
  <div class="row">
    <div class="col-md-4 col-lg-2">LEFT</div>
    <div class="col-md-8 col-lg-10">CONTENT</div>
  </div>
  
  В bootstrap ширина сайта (ширина сетки) делиться на 12 частей.
  Сетка состоит из контейнера (<div class="row"...) и вложенных элементов - колонок.
  Bootstrap позволяет создавать сайты, адаптированные под десктопные и мобильные браузеры за счет "перегрузки" классов сетки.
  Чтобы элемент стал колонкой ему необходиио задать класс вида "col-<mode>-<size>" (для всех колонок одной сетки сумма <size> должна равняться 12).
  Часть класса <mode>:
    - md - (middle) класс применяется для экранов средних размеров
    - lg - (large) класс применяется для жкранов больших размеров
